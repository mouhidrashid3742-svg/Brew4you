## 🎯 DEPLOY ON VERCEL (2 minutes)

### **STEP 1: Go to Vercel Dashboard**
1. Open: **https://vercel.com/dashboard**
2. Click **"Add New..."** → **"Project"**

### **STEP 2: Import GitHub Repository**
1. Click **"Continue with GitHub"** (if prompted to authorize)
2. Search for: **`Brew4you`**
3. Click **"Import"**

### **STEP 3: Configure Project**
1. **Project Name:** `brew4you` (or your choice)
2. **Framework Preset:** Select **"Next.js"** (auto-detected)
3. **Build & Development Settings:** Leave defaults
4. Click **"Continue"**

### **STEP 4: Add Environment Variables**

Click **"Environment Variables"** and add these 5:

```
Name: MONGODB_URI
Value: <YOUR_MONGODB_URI>
Select: Production, Preview, Development
Click: Add

Name: ADMIN_SECRET
Value: <ADMIN_SECRET>
Select: Production, Preview, Development  
Click: Add

Name: NEXT_PUBLIC_ADMIN_SECRET
Value: <ADMIN_SECRET>
Select: Production, Preview, Development
Click: Add

Name: NEXT_PUBLIC_WHATSAPP
Value: 923205950705
Select: Production, Preview, Development
Click: Add

Name: NEXT_PUBLIC_PHONE
Value: +923205950705
Select: Production, Preview, Development
Click: Add
```

### **STEP 5: Deploy! 🎉**
1. Click **"Deploy"**
2. Wait 2-3 minutes for deployment
3. ✅ **YOU'RE LIVE!**
