## 🚀 QUICK START (5 MINUTES)

### To Change a Price:
1. Open: `data/menu.ts`
2. Find: Item name (Ctrl+F)
3. Change: The price number
4. Save: Ctrl+S
5. Done! ✅

### To Add a New Item:
1. Open: `data/menu.ts`
2. Go to: End of file
3. Copy: Template from guide
4. Fill: Your item details
5. Save: Ctrl+S
6. Done! ✅
